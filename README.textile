h1. Setup

@wget http://downloads.typesafe.com/play/2.2.1/play-2.2.1.zip@
@unzip play-2.2.1.zip@
@./play-2.2.1/play test@

Replace 'test' with other Play commands, e.g. 'run' or 'eclipse' (to generate Eclipse project config files).
